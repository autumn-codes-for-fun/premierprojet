# Premierprojet

## Description

Ce projet est un **site vitrine** destiné à présenter mes œuvres à l’aquarelle.

Le but principal de ce projet était de **m’entraîner aux bases du développement web** (HTML, CSS et JavaScript) en créant un site à la fois **dynamique** et **interactif**. Il s’inscrit dans une démarche d’apprentissage progressif et constitue une base réutilisable pour de futurs projets, notamment dans la **création de formations digitales** . C'est aussi unne première approche de **l'expérience utilisateur**.

Le dossier **visuel** du projet contient des **captures d’écran du site**, permettant d’avoir un aperçu du rendu final sans lancer l’application.

## Technologies utilisées

* **HTML5** : structure des pages
* **CSS3** : mise en forme et design
* **JavaScript** : interactions et dynamisme du site

## Arborescence du projet

Voici l’arborescence générale du projet :


> ⚠️ Certains fichiers images sont encore manquants


## Installation

1. Télécharger l’intégralité des fichiers du projet
2. Placer tous les fichiers dans un **même dossier** en conservant l’arborescence
3. Ouvrir le fichier `accueil.html` dans un navigateur web

## Objectif pédagogique

Ce projet m’a permis de :

* Comprendre la structure d’un site web
* Manipuler le DOM avec JavaScript
* Gérer un projet simple avec une organisation claire des fichiers

Il constitue une première étape solide dans mon apprentissage du développement web.
